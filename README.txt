## Chatoon

Matthieu Vignolle & Thibault Germain - Development student at [Webstart][1].


----------

### Synopsis

This study project was to build a chat with socket.io. We are using HTML, CSS, Javascript and Node.js.
    
### Installation

First, download the zip repository by [clicking here][2] or clone it with your terminal by using this link: ```git@github.com:Thibgr/chatoon.git```

Then open ```index.html``` in Google Chrome by preference or Safari. It might have some bugs with other browser.
